Библиотека для упрощения создания видеоконтейнера для плеера Яндекс

yavideocontainer.swc -- библиотека, включайте её в свой проект
asdoc -- документация по библиотеке
test-implementation -- простейшая реализация API видеоконтейнера
dummy-player -- простейший плеер, загружающий видеоконтейнер. Клавишами 1-9 на клавиатуре можно переключать качества (если ваш видеоконтейнер позволяет переключать качества)
